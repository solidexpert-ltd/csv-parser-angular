## [1.0.0] - 2025-01-27

### Added
- Initial release of CSV Import Angular library
- Support for CSV, TSV, XLS, and XLSX file formats
- Modal and inline modes
- Dark and light theme support
- Internationalization (i18n) with English, Spanish, French, and German translations
- Smart column mapping with auto-matching
- Template-based column validation
- Required field validation
- Download template functionality
- Custom styling support
- Custom primary color theming

### Features
- Step-by-step import wizard (Upload → Select Header → Map Columns → Complete)
- Drag and drop file upload
- Row selection for header detection
- Column mapping interface
- Import result data structure
